Arquitetura da Landing Page — 3 Abordagens

Escolha a narrativa visual que melhor comunica o valor do ScrapJobs

A

Narrative Scroll (Recomendado)

Inspiração: Vercel, Linear

Cada secção ocupa ~80-100vh, criando uma "história" enquanto o utilizador faz scroll. Transições suaves entre secções com Framer Motion whileInView.


Fluxo:

  1. 🔥 A Dor — Full-screen, fundo escuro, tipografia gigante, texto provocativo que ocupa o ecrã inteiro
  2. 🎯 Killer Features (Zig-Zag) — 3 blocos alternados (texto esquerda/UI direita, depois inverte). Cada bloco tem parallax leve no UI snippet
  3. ⚡ Fluxo 1 Minuto — 3 steps horizontais com linha conectora animada, números grandes
  4. 🏢 Social Proof — Carrossel existente com novo título "Monitoramos as empresas que você ama"

Prós

  • Máximo impacto emocional — história contínua
  • Scroll-triggered animations criam "wow factor"
  • Padrão premium SaaS (Vercel, Stripe)

Contras

  • Mais pesado em animações (mitigado com Framer Motion lazy)
  • Pode ser longo em mobile
B

Bento Grid Compacto

Inspiração: Apple, Raycast

Após a secção da Dor (full-width), as features são apresentadas num Bento Grid (2x2 ou 3-col assimétrico). Cada célula é um "UI snippet" clicável.


Fluxo:

  1. 🔥 A Dor — Igual à Abordagem A
  2. 🎯 Features como Bento Grid — Grid assimétrico: célula grande para "O Radar" (scraping), duas menores para "ATS" e "PDF". Cada célula tem o UI snippet interativo dentro
  3. ⚡ Fluxo 1 Minuto — Integrado dentro do grid como uma row horizontal abaixo
  4. 🏢 Social Proof — Logo strip abaixo do grid

Prós

  • Extremamente escaneável — tudo visível de uma vez
  • Visual muito moderno (Apple WWDC style)
  • Menos scroll = mais rápido para converter

Contras

  • Menos espaço para copy persuasivo por feature
  • Zig-zag com texto longo não funciona em grid
  • Difícil acomodar 3 blocos de texto detalhado
C

Híbrido (Dor + Zig-Zag + Bento Steps)

Inspiração: Stripe, Automarticles

Combina o melhor das duas: Dor full-screen, Features em zig-zag clássico (com mais espaço para copy), e o Fluxo 1 Minuto como mini Bento de 3 cards.


Fluxo:

  1. 🔥 A Dor — Full-screen escuro, tipografia impactante
  2. 🎯 Killer Features (Zig-Zag puro) — 3 rows alternadas. Texto extenso de um lado, UI snippet premium do outro. Separadas por espaço generoso
  3. ⚡ Fluxo 1 Minuto (Bento 3-col) — 3 cards lado a lado com ícone, número e descrição curta. Visual de "receita rápida"
  4. 🏢 Social Proof — Carrossel com título atualizado

Prós

  • Zig-zag dá espaço máximo ao copy agressivo
  • Steps em bento = escaneável e rápido
  • Mais alinhado com o briefing (Automarticles)

Contras

  • Página mais longa que o Bento puro
  • Precisa de breakpoints mobile bem trabalhados
